## Description
This website used for counting a loan interest in IDR only. There is 2 card box they are: 
- first is for flexible counting,
- second is interest rate changed by it ammount and counted due to month

The formula for counting interest is below:

$P + \frac{P \cdot \left( \frac{r}{100} \right) \cdot t}{n}$

<div align='left'>
$$P + \frac{P \cdot \left( \frac{r}{100} \right) \cdot t}{n}$$
</div>

P = total loan amount
r = interest rate
t = total year loan (double)
n = total month loan
